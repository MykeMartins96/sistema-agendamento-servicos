# 📅 Sistema de Agendamento de Serviços

## 📖 Sobre o Projeto

O sistema permite que clientes realizem agendamentos de serviços e que administradores visualizem e gerenciem os atendimentos por meio de um painel administrativo.

---

## 🚀 Funcionalidades

### Área do Cliente

- Agendamento de serviços
- Cadastro de nome, telefone, serviço, data e horário
- Consulta de horários disponíveis
- Confirmação do agendamento

### Área Administrativa

- Login administrativo
- Visualização dos agendamentos
- Consulta dos dados dos clientes
- Alteração do status dos atendimentos
- Exclusão de agendamentos
